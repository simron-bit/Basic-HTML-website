# Basic-HTML-website
A basic html only website with multiple pages

## Details about the website
1. It is a basic personal website created using html semantic tags
2. All the pages contain a nav bar that contain hypertexts for pages: homepage, projects, articles and contacts

## What does this project focus on?
1. This projects focuses on creating a website using html only
2. Linking multiple pages within the website and focusing on the structure rather than the style

## How to run the project?
1. Clone the repository on your local device
2. Click on the index.html page and open it in your browser
